🧪 Job Application Tracker - Test Guide

📋 Prerequisites

Make sure you have:

  • ✅ Node.js installed (version 14 or higher)
  • ✅ The Job Application Tracker project running
  • ✅ A modern web browser (Chrome, Firefox, Safari, Edge)

🚀 Step 1: Start the Application

If the application is not already running, start it with:

npm start

The application should open at http://localhost:3000

📊 Step 2: Open the Application

Click the button below to open the Job Application Tracker:

Open Job Application Tracker

🎯 Step 3: Add Sample Data

To test the application with realistic data:

  1. Open the browser's Developer Tools (Press F12 or right-click → Inspect)
  2. Go to the Console tab
  3. Copy the entire content from sample-data.js
  4. Paste it into the console and press Enter
  5. Run the command: addSampleData()
Note: This will clear any existing data and replace it with sample job applications.

✅ Step 4: Verify Sample Data

After running the sample data script, you should see:

  • ✅ 8 job applications from top tech companies
  • ✅ Different statuses: Applied, Interviewing, Offered, Rejected, Withdrawn
  • ✅ Complete information including contact details and descriptions
  • ✅ Various job types and salary ranges

🧪 Step 5: Test All Features

Now you can test all the application features:

📊 Dashboard Testing

  • ✅ View all job applications in cards
  • ✅ Status statistics (click to filter)
  • ✅ Search by company name or position
  • ✅ Filter by application status
  • ✅ Sort by date, company, position, or status
  • ✅ Export data to JSON file
  • ✅ Import data from JSON file

➕ Add Job Testing

  • ✅ Fill out the comprehensive form
  • ✅ Test form validation (required fields)
  • ✅ Add contact information
  • ✅ Include job description and notes
  • ✅ Set application status and job type
  • ✅ Save and return to dashboard

📝 Job Details Testing

  • ✅ View detailed job information
  • ✅ Click contact links (email, phone)
  • ✅ Edit job information inline
  • ✅ Delete job with confirmation
  • ✅ View timestamps (created/updated)

🔧 Troubleshooting

If you encounter any issues:

  • Make sure the React app is running on http://localhost:3000
  • Check that the browser console is open (F12)
  • Ensure you copy the entire sample-data.js content
  • Try refreshing the page if the data doesn't appear
  • Check the console for any error messages
